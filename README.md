# Portfolio Website

A personal portfolio website showcasing my skills, projects, and contact information as a web developer.

## Description

This is a responsive portfolio website built with HTML, CSS, and JavaScript. It features a modern design with sections for home, profile, skills, projects, and contact information. The website is designed to highlight my work and provide an easy way for potential collaborators or employers to get in touch.

## Features

- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Smooth Navigation**: Fixed navigation bar with smooth scrolling to sections
- **Profile Section**: Personal information with a circular profile photo
- **Skill Showcase**: Display of technical skills with icons
- **Project Gallery**: Showcase of completed projects with descriptions
- **Contact Information**: Direct links to email, GitHub, and LinkedIn
- **Modern UI**: Clean, professional design with animations and hover effects

## Technologies Used

- HTML5
- CSS3 (with custom properties for theming)
- JavaScript (ES6+)
- Tailwind CSS (via CDN for utility classes)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/michaeldsnto/portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```

3. Open `portofolio.html` in your web browser.

## Usage

Simply open the `portofolio.html` file in any modern web browser. The website is fully self-contained and doesn't require a web server for basic viewing.
